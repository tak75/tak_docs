============
VisualStudio
============

ショートカット
==============

パラメータヒント
  Ctrl + Shift + Space

インデント整形
  Ctrl + K -> Ctrl + F

ブロック終端移動
  Ctrl + ]

全画面表示
  Alt + Shift + Enter

ビルド
  Ctrl + Shift + B

コードスニペット
=================

構文を入力し、Tabを2回押すことで雛型を入力してくれる

リファクタリング
=================

フィールドのカプセル化（フィールドをプロパティに変更し、変更したプロパティを使用するように更新する）
  Ctrl + R -> Ctrl + E

リネーム
  Ctrl + R -> Ctrl + R
