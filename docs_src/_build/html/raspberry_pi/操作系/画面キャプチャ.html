画面キャプチャ¶

scrot¶

以下にてインストールする:

sudo apt-get install scrot

使用方法は、

  • 画面全体:

    scrot

  • マウスで指定したウィンドウの中:

    scrot –s

  • マウスで指定したウィンドウ枠ごと:

    scrot –s -b
