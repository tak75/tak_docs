メソッドリスト¶

メソッド（シーケンスは作成しない）¶

メソッド

説明

All

全ての要素が条件を満たすかを判定する。引数に条件を書ける

Any

条件を満たす要素が1つでもあるかを判定する。引数に条件を書ける

Contains

要素に指定した値が存在するかを調べる

Count

シーケンスの要素数を返す。引数に条件を書ける

LongCount

シーケンス内の要素の数をlong型で返す。 Count メソッドでは扱えないほど大きなシーケンスを扱う場合に使う

TryGetNonEnumeratedCount

列挙を行わずにシーケンスの要素の数を調べられるかやってみる。.NET6以降

ElementAt

シーケンスの指定インデックスの要素を取得する

ElementAtOrDefault

シーケンスの指定インデックスの要素を取得する。存在しない要素を指定した場合に例外ではなく型のデフォルト値を返す

ForAll

Single

たった1つのが該当データを取り出す。存在ない場合、2つ以上存在する場合は例外

SingleOrDefault

たった1つのが該当データを取り出す。存在しない場合は既定値。2つ以上存在する場合は例外

SequenceEqual

2つのシーケンスの各要素が等しいことをチェックする

WithCancellation

WithDegreeOfParallelism

WithExecutionMode

WithMergeOptions

ファクトリメソッド¶

メソッド

説明

Empty

空のシーケンスを作成する

Repeat

同じ値が指定回数繰り返されるシーケンスを生成する

Range

要素の値が連番の配列やリストを作成する

IEnumerable<T>からコレクションへの変換¶

メソッド

説明

ToArray

シーケンスから配列を作成する

ToDictionary

シーケンスからDictionaryを作成する

ToHashSet

シーケンスからHashSet(重複したオブジェクトを追加できないリストクラス)を作成する

ToList

シーケンスからListを作成する

ToLookup

GroupBy メソッドと同機能＋キーを指定して対象項目の要素のみを取り出せる。hogeArr["青"]など

型変換¶

メソッド

説明

Cast

シーケンスの個々の要素の型変換を行う。変換できない場合は例外発生（⇔ OfType）

OfType

シーケンスの個々の要素の型変換を行う。変換できない場合は無視（⇔ Cast）

フィルタリング¶

メソッド

説明

Where

条件に適合する要素としない要素を識別し、条件に適合する要素だけからなるシーケンスを生成する

Distinct

シーケンスから重複する要素を取り除き、どの値も1つしか含まれないようにする

DistinctBy

Distinctメソッドと似ているが、等価を判定する値を選ぶキーセレクターを指定する点で異なっている

First

シーケンスの最初の要素、または条件を満たす最初の1つを返す。該当データがない場合は例外となる

FirstOrDefault

シーケンスの最初の要素、または条件を満たす最初の1つを返す。該当データがない場合は型の既定値を返す

Last

シーケンスの最後の要素、または条件を満たす最後の1つを返す。該当データがない場合は例外となる

LastOrDefault

シーケンスの最後の要素、または条件を満たす最後の1つを返す。該当データがない場合は型の既定値を返す

Skip

シーケンスの先頭のn個の要素を読み飛ばし、新たなシーケンスを得る

SkipLast

シーケンスの末尾のn個の要素を読み飛ばし、新たなシーケンスを得る

SkipWhile

シーケンスにおいて、特定の条件を満たす要素が続く限り読み飛ばし、新たなシーケンスを得る

Take

シーケンスの先頭のn個の要素を取得し、新たなシーケンスを得る

TakeLast

シーケンスの末尾のn個の要素を取得し、新たなシーケンスを得る

TakeWhile

シーケンスにおいて、特定の条件を満たす要素が続く限り取得し、新たなシーケンスを得る

単一シーケンスの加工¶

メソッド

説明

Select

シーケンスを加工して新しいシーケンスを作成する

SelectMany

シーケンスの各要素をIEnumerable<T>に射影し、結果のシーケンスを1つのシーケンスに平坦化する

Append

配列やリストの最後に指定した要素を追加し、新たな別の配列やリストを作成する

Prepend

配列やリストの先頭に指定した要素を追加し、新たな別の配列やリストを作成する

Chunk

シーケンスを指定個数で分割する

Order

シーケンスを昇順に並び替える。キーセレクターなし。.NET7以降

OrderBy

シーケンスを昇順に並び替える。キーセレクターあり

OrderDescending

シーケンスを降順に並び替える。キーセレクターなし。.NET7以降

OrderByDescending

シーケンスを降順に並び替える。キーセレクターあり

ThenBy

OrderBy メソッドでソートした結果に、さらに条件を追加してソートする

ThenByDescending

結果が降順となる以外、 ThenBy メソッドと同じ

Reverse

シーケンスの順番を逆転させる

DefaultIfEmpty

シーケンスが空の場合はデフォルト値のみを含むシーケンスを返す。空ではない場合はシーケンスをそのまま返す

シーケンスの合成¶

メソッド

説明

Concat

2つのシーケンス（(A1 A2 A3) (B1 B2 B3 B4 B5)）を単純に連結する。A1 A2 A3 B1 B2 B3 B4 B5

Zip

2つのシーケンス（(A1 A2 A3) (B1 B2 B3 B4 B5)）を同一番号の要素を1つの組として連結する。(A1 B1) (A2 B2) (A3 B3)

シーケンスの合成（集合）¶

メソッド

図

説明

Union

和集合(OR)。2つのシーケンスを連結する。その際、重複する要素は削除する

UnionBy

キー セレクター関数を使用するだけでUnionと同じ機能

Intersect

積集合(AND)。2つのシーケンスに共通して含まれる要素のシーケンスを作成する

IntersectBy

キー セレクター関数を使用するだけでIntersectと同じ機能

Except

2つのシーケンスにおいて、片方のシーケンスのみに存在する要素のシーケンスを作成する

ExceptBy

キーセレクター関数を使用するだけでExceptと同じ機能

シーケンスの合成（内部結合、外部結合）¶

メソッド

図

説明

Join

内部結合。2つのシーケンスの要素を、指定キーによって1つにまとめる

GroupJoin

外部結合。2つのシーケンスのキーが等しいかどうか判定し、要素を関連づけて結果をグループ化する

算術演算¶

メソッド

説明

Aggregate

String.Join()と似た挙動。要素同士を指定フォーマットで結合。要素が数値の場合は指定フォーマットで集計できる

Average

平均値を計算する。nullが含まれる場合はその要素を無視した平均値が計算される

Sum

入力シーケンスの合計を計算する

Max

シーケンスの最大値を得る。最大値を提供したオブジェクトではなく最大値を返す

MaxBy

キーセレクターを指定でき、キー値の最大を調べる。最大値を提供したオブジェクトそのものを返す

Min

シーケンスの最小値を得る。最小値を提供したオブジェクトではなく最小値を返す

MinBy

キーセレクターを指定でき、キー値の最小を調べる。最小値を提供したオブジェクトそのものを返す

グループ化¶

メソッド

説明

GroupBy

指定した要素でグループ化する

GroupByDescending

結果が降順となる以外、 GroupBy メソッドと同じである

並列クエリ¶

メソッド

説明

AsParallel

クエリの並列化を有効にする。通常のクエリを並列クエリに変換する

AsEnumerable

AsSequential

並列クエリから直列クエリに変換する。 AsEnumerable と同じ？

AsOrdered

並列クエリにおいて、順番を守る

AsUnordered
