文字列補間¶

  • 先頭に「$」記号が付いた文字列

  • その中の波かっこ「{}」で囲まれた部分の式を評価して文字列に当てはめてくれる

    Text1.Text = $"現在時刻は{DateTimeOffset.Now:HH:mm}";
// Text1.Text = 「現在時刻は12:00」などとなる
