Sphinxドキュメントの公開方法¶

ディレクトリ構成¶

github_sphinx_example
├── .gitignore
├── docs
│   ├── .nojekyll（Jekyllを使用しないようにするために必要）
│   ├── 公開するhtmlファイル
└── docs_src
    ├── build元のrstファイル

.nojekyllファイルの作成方法¶

コマンドプロンプトにて、以下入力でOK:

type null > .nojekyll
