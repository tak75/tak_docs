pigpio 有効化方法¶

  1. 図 4 のように、リモートGPIOを有効に設定する

  2. pigpio デーモンを有効化する:

    sudo systemctl enable pigpiod

  3. 念のため、pigpio デーモンの状態を確認する:

    systemctl status pigpiod
# runnning状態となっていればOK
