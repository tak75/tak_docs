Raspberry Pi Desktop インストール方法¶

Raspberry Pi Desktop インストール¶

  • ここ からisoファイルをダウンロード
  • VirtualBox で新規に Virtual Machine を作成しインストール（詳細は 参考 を参照）

警告

Oracle VM VirtualBox マネージャー -> 設定 -> 一般 -> 基本 -> バージョン については、「Debian (64-bit)」ではなく、isoファイルに合わせて「Debian (32-bit)」を選択すること。 そうしないと、スクリーンサイズ変更やクリップボード共有ができなくなる。

インストール後の処理¶

  • パッケージ更新:

    sudo sh -c 'apt update && apt upgrade -y && reboot'

VBoxLinuxAdditionsインストール¶

本インストール処理は、スクリーンサイズを変更したり、クリップボードの共有を行うために必要。

  1. ディスク挿入:

    VirtualBoxのMenu: Devices > Insert Guest Additions CD Images...

注釈

既に挿入されている場合はエラーが出る。その場合は次のステップへ進む。

  1. 仮想機内でターミナルを開いて、コマンド実行:

    sudo bash /media/cdrom/VBoxLinuxAdditions.run

  2. 待つ

  3. 一応、再起動:

    sudo reboot
